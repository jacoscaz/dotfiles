
DOTFILES
========

VIM
---

		$ ln -s /path/to/dotfiles/.vimrc /path/to/home/.vimrc
		$ ln -s /path/to/dotfiles/.vim /path/to/home/.vim

TMUX
----

    $ ln -s /path/to/dotfiles/.tmux.conf /path/to/home/.tmux.conf

LOCALE
------

    $ vim ~/.bash_profile

Add following line:

    source /path/to/dotfiles/.bash-locale
